Methodology

Each benchmark is a matched pair under examples/ - the same workload implemented in Mettle and C, printing Time: <N> us via QueryPerformanceCounter.

Mettle builds use --build --emit-obj --linker internal --release. C builds use GCC with -O3 -lkernel32. Runtime ratios are MettleÃ·C on median microseconds; values below 1.0 mean Mettle is faster.

Parity is defined as within Â±5% of C. Geometric means aggregate ratios without letting a single outlier dominate the headline number.
