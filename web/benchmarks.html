Methodology

Test Environment

Benchmarks are run locally. Run the script to update results:

.\tools\benchmark\run-benchmarks.ps1
.\tools\benchmark\run-benchmarks.ps1 -BuildCompiler
  • Methlang: --release (optimizations, no runtime checks)
  • C: gcc -O2
  • Timing: GetTickCount64 (Windows) or gettimeofday (Linux)
  • Output: web/benchmarks.json (served at /benchmarks.json)
